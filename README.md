### install

#### with pnpm

```sh
pnpm add @jiangweiye/tsconfig -D
```

#### with yarn

```sh
yarn add @jiangweiye/tsconfig -D
```

#### with npm

```sh
npm install @jiangweiye/tsconfig -D
```

### use

> `base`

```json
{
    "extends": "@jiangweiye/tsconfig/tsconfig.base.json"
    // ...
}
```

> `node`

```json
{
    "extends": "@jiangweiye/tsconfig/tsconfig.node.json"
    // ...
}
```

> `vue`


> `react`
